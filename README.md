# EgTrainer

EgTrainer 是一个通过翻译例句学习英语的网页应用。用户先根据中文例句写出英文翻译，再与参考英文例句对比；应用支持 AI 评估、重点高亮、学习笔记、历史复习和 HTML 导出，适合把词典、语法书或本地例句库变成可持续积累的主动输出练习。

## 使用方式

核心目标是把「看例句」变成「用例句练表达」：

1. 选择例句库或导入本地JSON 文件。
2. 在 Topic 导航中选择学习主题或章节。
3. 根据中文例句输入英文翻译并提交。
4. 对比参考英文例句，比较语法、用词和表达差异。
5. 可调用 AI 对翻译进行评估，按需要标注重点并记录笔记。
6. 查看和复习已练习内容。
7. 导出学习记录。

## 主要功能

- **公开例句库**：首页可直接选择公开例句库。
- **本地例句库导入**：支持导入本地 JSON文件。
- **左侧导航**：快速浏览例句库章节，支持搜索。
- **翻译练习**：展示中文例句，用户输入英文翻译后提交。
- **对比翻译**：提交后展示用户翻译和参考英文例句。
- **高亮标注**：可对用户翻译或参考英文例句选中文本并添加绿色 / 红色标注。
- **学习笔记**：每条练习可记录语法点、用词差异、翻译技巧等笔记。
- **AI 评估**：自定义API key和提示词；支持 OpenAI、DeepSeek 或自定义兼容 Chat Completions 的 API 地址；AI 评估提示词支持 `{expression}`、`{chinese}`、`{userTranslation}`、`{reference}` 变量。
- **历史练习**：同一例句多次练习时，可查看历史提交记录。
- **日期复习**：支持复习今天、最近 7 天、最近 30 天或自定义日期范围内练过的内容。
- **导出记录**：将已提交的翻译、高亮、参考例句和笔记导出为 HTML 文件。
- **本地存储**：练习进度、翻译、标注、笔记和 AI 设置保存在当前浏览器的 `localStorage` 中。

## 当前例句库

当前公开例句库包含：

- **LLA Lexicon**：结合了《朗文写作活用词典》（Longman Language Activator, LLA）的词库，以及《朗文多功能分类词典》（Longman Lexicon of Contemporary English, LEXICON）的单词顺序。
- **Collins C4**：来自《柯林斯语法大全》第四章相关例句库，目前仍在完善中。

[例句库数据格式说明](dicts/Example_Import.md)

## 数据与隐私

- 本地导入的词典文件不会上传到服务器。
- API key、练习记录保存在当前浏览器。
- 清空练习记录只会删除当前浏览器保存的翻译、高亮、笔记、AI 评估和复习记录，不会删除词典文件或 AI 设置。
